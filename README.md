
# Hyp-Poi-Vor: Hyperbolic Poisson Voronoi Tesselations

This project is a Python workspace for research and experimentation with Poisson Voronoi tesselations in the hyperbolic plane.

## Structure
- `src/` — Source code for the project
- `tests/` — Unit tests
- `requirements.txt` — Python dependencies

## Setup
1. (Recommended) Create a virtual environment:
	 ```powershell
	 python -m venv .venv
	 .venv\Scripts\Activate.ps1
	 ```
2. Install dependencies:
	 ```powershell
	 pip install -r requirements.txt
	 ```

## Running Code
- Place your main scripts in `src/`.
- Run scripts with:
	```powershell
	python src\your_script.py
	```

## Running Tests
- Place your test files in `tests/` (e.g., `test_example.py`).
- Run tests with:
	```powershell
	python -m unittest discover tests
	```
